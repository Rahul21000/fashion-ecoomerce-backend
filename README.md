# Fashion eCommerce Backend (Java + Spring Boot)

A scalable backend system for a Fashion eCommerce platform built using Java, Spring Boot, and Microservices architecture.
This project handles user management, product catalog, orders, payments, and authentication.

# Features
👤 User Registration & Authentication (JWT-based)
🛒 Product Catalog Management
📦 Order Processing System
🔐 Secure APIs with Spring Security
📡 RESTful APIs for frontend/mobile apps

# Architecture

Auth Service → Login, JWT, Security
User Service → User Profile Management
Product Service → Product listing & search
Order Service → Order placement & tracking
Payment Service → Payment processing

# Tech Stack
Backend: Java, Spring Boot
Security: Spring Security, JWT
Database: MySQL
Build Tool: Maven
